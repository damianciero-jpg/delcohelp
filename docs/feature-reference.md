## AI Assistant

| Feature | Description |
|---|---|
| Conversational AI | Powered by Anthropic Claude — answers resource and benefits questions in plain language |
| Suggested prompts | "I need food near me tonight," "How do I apply for SNAP?", "I'm facing eviction" |
| Crisis escalation | Routes crisis questions to 988 and 911 |
| Daily message limit | 5 messages/day on free tier (resets at midnight) |
| Branded to your organization | "Ask SJC AI" / "Ask RPPC AI" on white-label apps |
| No conversation storage | Messages sent to Anthropic for processing; not stored |
